Mail Server

Mail server adalah program daemon yang bekerja menampung dan mendistribusikan email dalam suatu jaringan.

VOIP Server

Voice over Internet Protocol adalah teknologi yang memungkinkan percakapan suara jarak jauh melalui media internet.

DNS Server

DNS (Domain Name Server) adalah server yang digunakan untuk mengetahui IP Address suatu host lewat host name-nya.

DHCP Server

DHCP adalah protokol yang berbasis arsitektur client/server yang dipakai untuk mempermudah pengalokasian IP Address pada satu jaringan.

Web Server

adalah sebuah software yang memberikan layanan berbasis data dan berfungsi menerima permintaan dari HTTP atau HTTPS pada klien.

NTP Server

NTP adalah sebuah protokol yang berfungsi untuk menyamakan waktu dari setiap komputer yang terhubung

FTP Server

FTP server adalah suatu server yang menjalankan software yang berfungsi untuk memberikan layanan tukar menukar file dimana server tersebut selalu siap memberikan layanan FTP.

SSH Server

adalah suatu software yang berfungsi untuk menerima request dari client, yang bertugas mendeskripsi enkripsi client dan mengjalankan printah yang di instruksikan client.

VPN

adalah sebuah teknologi komunikasi yang memungkinkan untuk dapat terkoneksi ke jaringan public dan menggunakannya untuk dapat bergabung dengan jaringan local.
